Help

How to start ?

  • Open phplangeditor in the Tools menu of your Firefox or Mozilla browser.
  • Click on Reference file to select the original translation.
  • If you aim to complete an existing translation, click on Translation file to select it.
  • On the tool left Pane, select the item you are going to translate.
  • You will see the original translation on the upper right pane. Write your own translation on the lower right pane, and validate it by clicking on save current item.
    Repeat the operation for all the items you need to translate.
  • Finally click on Save translation to record your work in the file of your choice.

Tips

  • Modify the charset of the 'Reference file' during a translation can generate duplicate entries in the item list.
  • Translation charset is used to open the 'Translation file' and to save your work.
  • Modify the charset of the 'Translation file' during a translation could replace some translations by the one present if the loaded translation file (if there is one).
  • Ctrl-enter is a keyboard shortcut to Select next item.
